1Buat Google Sheet baru di sheets.google.com. Nama tab biarkan default atau ubah ke Transaksi.
2Buka Extensions → Apps Script. Hapus semua kode yang ada, paste kode dari file apps-script.gs yang disertakan. Save.
3Deploy → New deployment. Type: Web App, Execute as: Me, Access: Anyone. Copy URL yang muncul.
4Atau langsung klik Pakai Offline — data nyimpen di browser, bisa export CSV kapanpun.
